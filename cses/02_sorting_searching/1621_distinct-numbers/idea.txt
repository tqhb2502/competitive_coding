Distinct Numbers - CSES 1621
https://cses.fi/problemset/task/1621

## Đề bài
- Cho một danh sách gồm n số nguyên. Hãy đếm xem trong danh sách có bao nhiêu giá trị phân biệt (distinct).
- Ràng buộc: 1 <= n <= 2*10^5, 1 <= x_i <= 10^9.

## Ý tưởng
- Số lượng giá trị phân biệt chính bằng kích thước của tập hợp (set) các phần tử, vì set tự động loại bỏ các phần tử trùng lặp.
- Ta đọc toàn bộ n số, đưa tất cả vào một hash set rồi lấy số phần tử còn lại làm đáp án.
- Cách khác tương đương: sắp xếp mảng rồi đếm số vị trí có giá trị khác với phần tử liền trước (đây là lý do bài nằm trong chuyên đề Sorting and Searching). Tuy nhiên dùng set đơn giản và đủ nhanh.
- Code C++ đọc từng giá trị `long long` và chèn ngay vào `unordered_set`; đặt
  trước dung lượng và hạ `max_load_factor` để giảm số lần rehash.

## Độ phức tạp
- Thời gian: O(n) trung bình khi dùng hash set (hoặc O(n log n) nếu dùng cách sắp xếp). Với n <= 2*10^5 đều rất nhanh.
- Bộ nhớ: O(n) để lưu các phần tử.
