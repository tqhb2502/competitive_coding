Hidden Integer - CSES 3112
https://cses.fi/problemset/task/3112

Tags: binary-search, monotone-predicate, search-on-answer

## Đề bài
Judge giấu một số nguyên x với 1 <= x <= 10^9. Ta phải đoán ra x qua các truy vấn.
- Truy vấn (query): in ra dòng "? y" với 1 <= y <= 10^9.
  Judge trả lời:
    + "YES" nếu y < x (tức là x lớn hơn y).
    + "NO"  nếu y >= x (tức là x nhỏ hơn hoặc bằng y).
- Trả lời cuối cùng: in ra dòng "! x" rồi kết thúc chương trình.
- Giới hạn số truy vấn (query limit): tối đa 30 query.
- Bắt buộc phải flush stdout sau mỗi dòng (dùng endl), nếu không judge sẽ
  không nhận được query và chương trình bị deadlock.

## Ý tưởng
Dùng binary search theo giá trị của x.
- Duy trì đoạn [lo, hi] chứa x, khởi tạo lo = 1, hi = 10^9.
- Bất biến (invariant): luôn có lo <= x <= hi.
- Trong khi lo < hi:
    + Chọn mid = (lo + hi) / 2 (chia lấy phần nguyên). Vì lo < hi nên
      lo <= mid <= hi - 1, do đó 1 <= mid <= 10^9 - 1: query luôn hợp lệ.
    + Query "? mid".
      * Nếu "YES" (mid < x) thì x >= mid + 1, đặt lo = mid + 1.
      * Nếu "NO"  (x <= mid) thì đặt hi = mid.
- Khi lo == hi thì chỉ còn một ứng viên duy nhất, đó chính là x, in ra "! lo".

Tính đúng đắn: mỗi bước giữ nguyên bất biến lo <= x <= hi. "YES" nghĩa là
x nằm trong [mid+1, hi] nên đặt lo = mid+1 an toàn; "NO" nghĩa là x nằm trong
[lo, mid] nên đặt hi = mid an toàn. Vì mid < hi khi lo < hi nên đoạn luôn co
lại và vòng lặp chắc chắn kết thúc.

## Độ phức tạp
Mỗi query giảm kích thước đoạn [lo, hi] đi ít nhất một nửa. Số phần tử ban đầu
là 10^9 nên số query tối đa là ceil(log2(10^9)) = 30 (vì 2^30 = 1073741824 > 10^9).
30 <= 30 = query limit, thoả mãn giới hạn. Thời gian chạy O(log(10^9)).
